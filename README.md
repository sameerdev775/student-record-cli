# Student Record CLI App

This is a menu-driven Python CLI application.

## Features
- Add student records
- Calculate percentage and grade
- Store data using file handling
- View all stored student records
- Handles invalid input safely

## How to Run
python student_record_cli.py

## Concepts Used
- Functions
- Loops
- Conditionals
- File I/O
- Exception handling


## Versions

### V1 — `student_record_cli.py`
Original CRUD CLI app.

### V2 — `student_record_cli_v2.py`
Refactored version with:
- structured data parsing
- search by name
- find topper
- delete student
- cleaner output format
